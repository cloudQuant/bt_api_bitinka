# bt_api_bitinka

Bitinka exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_bitinka
```

## Usage

```python
from bt_api_bitinka import BitinkaExchangeDataSpot, BitinkaRequestTickerData
```

## Features

- REST API support
- API key authentication (X-API-KEY header)
- Symbol format: `BTC-USD` or `BTC_USD` -> `BTC/USD`
